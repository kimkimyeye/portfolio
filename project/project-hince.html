Project

Overview

화장품 브랜드 '힌스' 리디자인 및 웹사이트 제작

브랜드의 컨셉을 잘 반영하며, 사용자가 상품을 쉽게 찾을 수 있는 인터페이스를 설계하고자 했습니다.

쇼핑몰 홈페이지에서 자주 사용되는 스와이퍼 플러그인을 다양하게 활용해 적용했습니다.

PC, 태블릿, 모바일에서 모두 적용 가능한 반응형 홈페이지로 제작했습니다.

Design

Concept

화장품 브랜드인 힌스의 상품이 잘 보이는 이미지 중심의 레이아웃과 함께,

힌스의 부드러운 분위기를 잘 나타내기 위해 뮤트톤의 차분한 컬러와 세리프 폰트를 사용했습니다.

Problem Solving

스와이퍼 플러그인이 왜 작동을 안할까?

기존 스타일을 가지고 있는 스와이퍼 플러그인을 사용할때 스타일링에 어려움이 있었고

기존 구조나 클래스명을 수정할 경우 스와이퍼가 잘 작동되지 않는 문제가 있었습니다.

개발자 도구를 통해 기존 스와이퍼의 네비게이션, 버튼, 페이지네이션의

클래스명과 설정 값들을 확인하면서 하나씩 수정해 나갔습니다.

반응형 웹사이트의 제작 순서 때문에 어려움을 겪음

pc버전으로만 먼저 작업했다가 나중에 모바일 버전도 디자인 후 추가했습니다.

이후에 pc와 모바일에 맞지 않는 뷰포트가 있어서 태블릿 버전도 제작했는데

미디어 쿼리를 pc, 모바일, 태블릿 순으로 작업하니 중복된 코드가 많아지고 제작이 어려웠습니다.

앞으로 반응형 웹사이트를 제작할 땐 일관된 순서대로 작업해야겠다는 생각이 들었습니다.

브랜드 이미지를 보여주는 영상 소스가 없을 때

collection section을 디자인했을 때는 각 컬렉션의 타이틀 리스트가 있고

타이틀을 호버 했을 때 컬렉션의 무드와 맞는 영상을 보여주는 형식으로 계획했으나,

적절한 영상 소스를 찾기 어려웠습니다.

대신 CSS 애니메이션을 통해 움직이는 모션을 영상처럼 표현하고,

리스트를 호버 하면 각 컬렉션의 사진을 보여주는 형식으로 수정했습니다.

이 과정에서 시행착오를 많이 겪었기에, 앞으로는 활용할 수 있는

소스를 잘 파악하고 제작해야겠다는 생각이 들었습니다.

Review

이번 프로젝트에서는 특정 브랜드를 선택하여 웹 디자인과 퍼블리싱을 진행했습니다. 

스와이퍼 플러그인을 처음 사용해 보았는데,

다양한 기능과 많은 옵션으로 홈페이지를 구성하기에 좋았습니다.

초기에는 시행착오가 있었지만, 처음에만 플러그인 사용법을 잘 숙지하면

웹사이트 제작에 큰 도움을 받을 수 있다는 걸 알게 되었습니다.

쇼핑몰을 디자인해서 이미지 소스가 많았는데, 디자인과 퍼블리싱 단계 모두

파일 정리가 잘 될수록 작업이 수월해짐을 배웠습니다. 

디자인 단계에서부터 계획이 꼼꼼하게 짜여진 섹션은 작업이 수월했고,

계획이 부족한 채로 퍼블리싱에 들어간 섹션은 진행이 느렸기 때문에 앞으로는

디자인 단계에서부터 구조와 인터렉션을 충분히 생각하면서 작업해야겠다는 생각이 들었습니다.
